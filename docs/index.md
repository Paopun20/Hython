# Hython Documentation

Welcome to the Hython documentation! Hython is a Python-inspired scripting interpreter built on Haxe, designed for embeddable and sandboxable game scripting.

## What is Hython?

Hython brings Python-style syntax and semantics to the Haxe ecosystem. It's a lightweight, dynamic scripting language that integrates seamlessly with Haxe projects‚Äîmaking it perfect for:

- **Game scripting** (e.g., modding frameworks like Psych Engine)
- **Modular code execution** within larger Haxe applications
- **Sandboxed scripting** where you need a safer, isolated execution environment
- **Python developers** transitioning to Haxe-based systems

## What Hython Is NOT

Hython is **not a full Python implementation**. It focuses on core features and syntax rather than mimicking every Python library and behavior. For advanced Python features, consider:

- Using actual Python with `PythonExecutor` for heavy lifting
- Implementing custom Haxe extensions for specific needs
