# Hython Interpreter (on rewrite stage)

Hython is a Python interpreter written in Haxe.

It's designed for:

- working on any platform you want
- working with Haxe/Haxeflixel projects (any Haxe project, actually)
- easy integration with Haxe projects

> This project is not bind to CPython, btw.

## Features

On the rewrite stage for stack-based bytecode interpreter

- [] Bytecode interpreter
- [] Better REPL
- [] Better error handling
- [] More Python features (like generators, async/await, etc.)
- [] Performance improvements
